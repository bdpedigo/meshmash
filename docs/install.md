---
title: Installation
---

=== "uv"

    ```bash
    uv add meshmash
    ```

=== "pip"

    ```bash
    pip install meshmash
    ```

To include sample dataset utilities:

=== "uv"

    ```bash
    uv add "meshmash[datasets]"
    ```

=== "pip"

    ```bash
    pip install "meshmash[datasets]"
    ```
