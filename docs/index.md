`meshmash` is a Python library for mesh analysis and feature extraction, with a focus
on spectral geometry (Heat Kernel Signatures), mesh splitting and agglomeration. It also has some general utilities I have found useful.

Currently, meshmash also includes some integration with [CAVE](https://github.com/CAVEconnectome) connectomics data. This will likely be moved to a separate package or made optional in the future.

It also exposes a CLI for running HKS feature extraction pipelines on mesh files.
